Giới thiệu về PHP 1/ Môi trường phát triển cho PHP - PHP cần có môi trường thì mới chạy được, sẽ khác với Frontend: - Môi trường phát triển Web với PHP cần 3 thành phần chính sau: + Web server: hiểu đơn giản là nơi đặt website lên, hosting chính là 1 webserver, trên webserver này sẽ cài đặt 1 trong 2 dịch vụ chính sau: Apache hoặc Nginx Với XAMPP đã cài đặt sẵn Apache, webserver chính là máy tính của bạn (localhost, 127.0.0.1) + PHP Engine: xử lý ngôn ngữ lập trình PHP. Khi cài XAMPP đã cài đặt luôn PHP Engine rồi + Database - Cơ sở dữ liệu: lưu trữ dữ liệu cho hệ thống, với PHP thường đi kèm với CSDL MySQL. Với 1 trang web viết bằng PHP thì Database là bắt buộc phải có, để trang web là web động - có tính tương tác với user - Khi cài XAMPP đã có luôn cả 3 thành phần trên, còn nếu ko muốn dùng XAMPP thì cần phải cài đặt thủ công cả 3 thành phần trên - XAMPP chỉ dùng cho việc code trên localhost, còn trên server thật thì sẽ ko dùng XAMPP - SAu khi cài xampp, mọi file/thư mục code PHP của bạn bắt buộc phải đặt tại đường dẫn sau: C:/xampp/htdocs/. Gõ trên trình duyệt url có dạng sau để chạy file: localhost/ 2 - Mô hình Client - Server: 3 - PHP là gì: + Hypertext Preprocessor + Là ngôn ngữ lập trình chạy trên nền WEb + Có rất nhiều phiên bản PHP, mới nhất là 7.5 + Là mã nguồn mở + Tích hợp dc nhiều CSDL như MySQL, SQL Server, Oracle ... nhưng thông dụng nhất là MySQL 4 - Chương trình PHP đầu tiên + Truy cập đường dẫn: C:/xampp/htdocs + Tạo 1 thư mục tên là php0720e_demo tại đường dẫn trên + Trong thư mục này tạo 1 file tên: demo.php
