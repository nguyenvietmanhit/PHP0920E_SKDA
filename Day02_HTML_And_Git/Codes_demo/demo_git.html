Kỹ năng thao tác với GIT (tiếp) - Lấy code từ 1 link git khác về máy tính (trong trường hợp máy tính của bạn chưa có code này) git clone + Demo lấy code từ link Git của lớp về thông qua lệnh trên - Sau khi clone thành công code từ link Git, trong trường hợp người chủ link Git đó có update thêm dữ liệu, thì để đồng bộ code trên máy tính của bạn với code trên link Git đó, sử dụng lệnh sau: git pull origin git pull origin master
